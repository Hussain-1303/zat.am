zat.am

Match the सङ्ख्या (Numbers)

Try to make all सङ्ख्या (numbers) equal by visiting them. Move using arrow keys. Press R to reset settings to defaults.

Gameplay & Features

  • Select which digit to change
    • Digit you move from
    • Digit you go to
  • Select how to change the digit
    • Increase the digit
    • Decrease the digit
  • If you visit a digit that is equal to the maximum digit, it is reset to minimum digit in increasing mode (or vice versa).
